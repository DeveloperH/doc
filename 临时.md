- 更换网络重试，比如使用 4g 网络

- 在设备中增加固定的 hosts（如：140.82.113.4 github.com）

## 一、国内 CMS 模板（最常用）

### 1. 易优 EyouCMS

- 特点：**响应式、简单好改、免费商用多**

- 推荐模板：

  - 响应式科技企业官网
  - 机械设备 / 仪器行业模板
  - 通用企业展示站

  

- 获取：易优官网模板库 → 筛选 0 元免费

### 2. PbootCMS

- 特点：轻量、标签简单、二次开发友好
- 推荐：全行业自适应企业模板、产品展示型
- 资源：第三方模板站大量免费 / 低价合集

### 3. 织梦 Dedecms

- 特点：资源最多，老站常用
- 提醒：**注意安全补丁**，优先选响应式 HTML5 模板

### 4. 帝国 CMS / 迅睿 CMS

- 适合：下载站、资讯站、企业重型站
- 模板：结构严谨，适合需要权限 / 栏目复杂的站点

------

## 二、国际通用 CMS 模板

### WordPress（最推荐）

- 免费优质主题：Astra、GeneratePress、Neve
- 特点：极轻量、SEO 友好、一键导入演示站
- 适用：企业官网、博客、外贸、小程序适配

------

## 三、按用途直接选

- 企业官网：易优 / Pboot / WP Astra
- 产品展示 + 表单：易优响应式模板
- 博客 / 资讯：WP + 轻量主题
- 外贸多语言：WP + 多语言插件

------

## 四、模板必备要求（必看）

1. **响应式**：手机 / 平板自动适配
2. **HTML5+CSS3**：加载快、SEO 友好
3. 无恶意代码、无加密后门
4. 支持后台可视化修改
5. 带基础页面：首页、关于、产品、新闻、联系我们
